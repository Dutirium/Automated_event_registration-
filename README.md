# Automated_event_registration-
Online Event Registration System with a Flutter-based verification app. The platform handles participant registration, email verification, secure Razorpay payments, and database management through a web portal. Uses the Flutter app for fast on-site verification, enabling seamless check-ins and eliminating manual entry processes during events.
# Online Event Registration System

A platform for managing student event registrations with verification via a Flutter mobile application.

## Features
- Event registration portal
- Student verification through mobile app
- Admin dashboard
- Real-time data handling
